Heap Sort

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. The algorithm works by first building a max heap from the given array, and then repeatedly extracting the maximum element from the heap and placing it at the end of the array. The process continues until the heap is empty and the array is sorted.

There are mainly two steps in the algorithm:

1. Build a Max Heap: Convert the given array into a binary heap, where the value of each parent node is greater than the values of its children.

2. Extract Maximum: Remove the root (maximum element) from the heap, place it at the end of the array, and then reheapify the remaining heap.

3. Repeat: Repeat the extraction and reheapify process for the remaining elements until the entire array is sorted.

    
      procedure heapSort(A)
        n = length(A)
        for i = floor(n/2) downto 0
          heapify(A, n, i)
        end for
        for i = n-1 downto 1
          swap A[0] with A[i]
          heapify(A, i, 0)
        end for
      end procedure

      function heapify(A, n, i)
        largest = i
        left = 2*i + 1
        right = 2*i + 2
        if left < n and A[left] > A[largest]
          largest = left
        end if
        if right < n and A[right] > A[largest]
          largest = right
        end if
        if largest != i
          swap A[i] with A[largest]
          heapify(A, n, largest)
        end if
      end function
